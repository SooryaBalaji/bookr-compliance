# Bookr Compliance Command Centre

Enterprise-grade regulatory compliance and automated corporate filing tracker.

## Tech Stack
- Backend: FastAPI (Python 3.12)
- Database: PostgreSQL 16
- Frontend: HTML/JS/CSS (Static)

## Getting Started
1. Run `docker-compose up` to start the backend and database.
2. Access the dashboard at `http://localhost:8000`.
